# expres-react-app

Ref: https://daveceddia.com/create-react-app-express-backend/

## Startup client

```
npm start
```

## Startup backend proxy

```
npm run start-backend
```

## Startup by pm2

```
pm2 start process.yml
```

## Nodemon

[Nodemon](https://github.com/remy/nodemon) can also be applied:

```
nodemon
```
